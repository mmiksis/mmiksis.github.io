Run the Installation Wizard

When the ownCloud prerequisites are fulfilled and all ownCloud files are installed, the last step to completing the installation is running the Installation Wizard.

Before you begin

It is best practice to protect the Installation Wizard through some form of password authentication or access control. If the installer is left unprotected when exposed to the public internet, there is the possibility that an unauthorized user could finish the installation and block you out. Ensure that only you or someone from your organization can access the web installer.

Procedure

  1. Point your web browser to http://localhost/owncloud.
  2. Enter your administrator username and password
  3. Click Finish Setup.

What to do next

You can now begin using your new ownCloud server. There are further configuration options to set up your ownCloud server for best performance and security. For detailed information, refer to the following:
  • Data Directory Location
  • Database Choices
  • Post-Installation Steps
